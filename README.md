# 🚀 Hackathon Survival Kit

A web-based toolkit that helps hackathon teams plan, organize, and execute their projects efficiently. Instead of spending hours deciding ideas and dividing work, the Hackathon Survival Kit provides smart recommendations, role assignments, development timelines, and presentation guidance—all in one place.

---

## 📌 Project Overview

Hackathons are exciting but often chaotic. Teams struggle with:

- Choosing the right project
- Identifying team strengths
- Dividing responsibilities
- Managing limited time
- Preparing presentations

Hackathon Survival Kit simplifies this process by providing an organized workflow from idea selection to final presentation.

---

## ✨ Features

### 👥 Team Skill Analysis
- Analyze available team skills
- Identify strengths and weaknesses
- Suggest missing skills
- Recommend improvements based on team composition

---

### 💡 Project Recommendation Engine
Get project ideas based on:

- Selected Domain
- Available Hackathon Duration

Displays:

- Project Title
- Difficulty Level
- Tech Stack
- Must-have Features
- Optional Features

---

### 🎯 Team Role Assignment

Add team members and automatically assign suitable roles based on their primary skills.

Suggested roles include:

- Frontend Developer
- JavaScript Developer
- Database Developer
- UI/UX Designer
- AI/ML Developer
- Presentation Lead

---

### ⏳ Development Timeline Generator

Generate a structured development schedule for:

- 24 Hours
- 48 Hours
- 7 Days

The generated timeline helps teams manage time effectively during hackathons.
